# Bookipi Desktop Notetaker

This repository is used for distributing the releases of **Bookipi Desktop Notetaker**.

## Overview

Bookipi Desktop Notetaker is a desktop application for capturing and managing notes. This repo hosts the published release artifacts and serves as the distribution channel for the app.

## Releases

Release builds are published here. To download the latest version, check the [Releases](../../releases) page.

## Support

For issues or feedback, please open an issue in this repository.
